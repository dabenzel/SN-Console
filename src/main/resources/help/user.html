Name

user - output current user name

Description

The user command normally outputs the current logged in user name.

However this command is not guaranteed to do that since it will output the user name which started the JVM of the SN-Console. But this name can be set to something different. For instance with the "-Duser.name=insert_user_name_here" command.

Syntax

user

Arguments

There are no arguments available for this command.
