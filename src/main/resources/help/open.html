Name

open - open a URI in associated program

Description

Opens the specified URI in associated program. That could either be a browser or an editor.

Whether the URI will be opened in a new browser or a browser tab will be determined by the browser preferences.

Note that this command will respect any pop-up blocker settings of the default browser.

Also note that in case the command fails to open the specified URI it will not output any feedback of the cause. It is very likely that the URI simply could not be found.

Syntax

open <'uri'>

Arguments

<'uri'>

Represents the URI to open.
