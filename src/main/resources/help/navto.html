Name

navto - changes the working directory

Description

The navto command changes the current working directory.

This command can be used by either specifying the absolute path of the directory to change the working directory to, just providing the name (this only works if the directory is in the current working directory) or using the '..' operator which will go back one directory.

Syntax

navto [<'path'>]
navto [.]
navto [..]
navto [~]

Arguments

<'path'>

The path of the directory.

[.]

Use this argument to go back to the directory where the SN-Console has been started.

[..]

Use this argument to go back one directory.

[~]

Use this argument to go to the user directory.
